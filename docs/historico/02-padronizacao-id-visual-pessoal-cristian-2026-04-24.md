# Replicacao do design system da ID Visual Pessoal Cristian

- **Data:** 2026-04-24
- **Autor:** Codex
- **Escopo:** design-system / documentacao

## Contexto / Motivacao
A base visual pessoal do Cristian foi consolidada primeiro no `blog-cristian` e deve ser compartilhada com os demais projetos pessoais para manter consistencia de marca.

## O que mudou
- `docs/design-system/design-system.md` foi atualizado para a mesma especificacao da **ID Visual Pessoal Cristian** aplicada no `blog-cristian`.
- Mantidos os mesmos principios:
  - tokens, tipografia, paleta e componentes padronizados;
  - alteracao de marca restrita a logos, favicon e imagens pessoais.

## Impacto
- O `bio-cristian` passa a seguir o mesmo guia visual-base dos outros projetos pessoais.
- Reduz risco de inconsistencias de UI e acelera manutencao de design entre repositorios.
