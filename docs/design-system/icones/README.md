# Icones

Guarde aqui icones proprios da marca ou icones customizados usados no projeto.

## Formatos recomendados
- `.svg` (preferido).
- `.png` apenas quando SVG nao for viavel.

## Nomenclatura (ASCII + kebab-case, sem acentos)
- `icone-<nome>.svg` (ex.: `icone-seta-direita.svg`, `icone-menu.svg`).
- Para conjuntos/estados: `icone-<nome>-<estado>.svg` (ex.: `icone-coracao-preenchido.svg`).

Se o projeto usa uma biblioteca de icones (Lucide, Heroicons, etc.), documente em `design-system.md` qual biblioteca e padrao e use esta pasta apenas para icones customizados.
