# Favicon

Guarde aqui os arquivos de favicon e icones de app.

## Formatos recomendados
- `favicon.ico` (multi-resolucao, 16x16 + 32x32 + 48x48).
- `favicon.svg` (vetorial, para navegadores modernos).
- `favicon-16x16.png`, `favicon-32x32.png`.
- `apple-touch-icon.png` (180x180).
- `android-chrome-192x192.png`, `android-chrome-512x512.png`.
- `site.webmanifest` (quando aplicavel).

## Nomenclatura (ASCII + kebab-case, sem acentos)
Use exatamente os nomes canonicos acima — sao os que gerador de favicons/navegadores esperam.
